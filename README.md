# NetStone ![Nuget](https://img.shields.io/nuget/v/MinSharp)

NetStone is a portable and modern .NET FFXIV Lodestone API.

## What works

- [x] Characters
- [x] Character Search
- [x] FCs
- [x] FC Search
- [ ] PvP Teams
- [ ] PvP Team Search
- [ ] Linkshell
- [ ] Linkshell Search
- [ ] CWLS
- [ ] CWLS Search

Eorzea DB support is not planned.
